# Echo Lab (메타 구성용 폴더)

이 폴더는 에코 서버/클라이언트 구현을 위해 나중에 실제 소스코드를 넣는 구조 전용 위치입니다.

권장 배치 (구현 예정):
- echo_server.c : 에코 서버 진입점
- echo_client.c : 에코 클라이언트 진입점
- echo_common.h/.c : 에코 패킷/유틸 공통 함수(선택)

현재는 메타 정리 전용으로 비워 둔 상태입니다.
추후 아래 두 명령으로 빌드/실행 타깃을 붙이면 됩니다.

- echo 폴더에서: make server
- echo 폴더에서: make client
